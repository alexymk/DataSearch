<h1 align="center">Installation</h1>

<p>To install and configure DataSearch, follow these steps:</p>

<a href="https://github.com/figwix/DataSearch/archive/refs/tags/v1.0.zip">Download "DataSearch.zip" Here</a>

<p>
1 - Download the .zip folder.
2 - Unzip the folder.
3 - Launch “setup.py” in your terminal.
Or
1 - Open a terminal.
2 - Run “git clone https://github.com/figwix/DataSearch.git”
3 - Navigate into the folder “cd DataSearch”
4 - Run “python setup.py”
<br><br>
</p>
